#  Photo Editor Studio Pro

### 🔗 [Ver Aplicación en Vivo](https://facundoconsultas.github.io/editor-fotos-pro/)

Este es un editor de imágenes de alto rendimiento desarrollado con **JavaScript puro (Vanilla JS)** y **HTML5 Canvas API**. La aplicación permite a los usuarios cargar imágenes locales, aplicar ajustes técnicos de imagen en tiempo real y exportar el resultado final.

##  Desafíos Técnicos Superados
- **Manipulación de Gráficos (Canvas API):** Gestión del contexto 2D para renderizar filtros complejos directamente en el motor gráfico del navegador.
- **Procesamiento Local (FileReader API):** Implementación de lectura de archivos asíncrona para que las fotos se procesen en el cliente, garantizando privacidad y velocidad.
- **Sincronización del DOM:** Vinculación de estados de sliders dinámicos con los algoritmos de filtrado de imagen.
- **Exportación de Blobs:** Conversión de datos de pixeles en archivos descargables de formato `.png`.

##  Funcionalidades
- **Filtros Dinámicos:** Brillo, Contraste, Saturación, Sepia y Desenfoque (Blur).
- **Preview en Tiempo Real:** Visualización instantánea de los cambios realizados.
- **Sistema de Reset:** Restauración de valores originales con un solo clic.
- **Descarga Directa:** Función para guardar la imagen editada en el dispositivo.

##  Tecnologías
- **HTML5 & CSS3** (Estructura Semántica y Diseño Responsivo)
- **JavaScript Moderno (ES6+)** (Canvas API, FileReader, Event Listeners)
- **Google Fonts** (Tipografía Inter para interfaz moderna)
